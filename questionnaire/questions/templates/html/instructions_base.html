{% trans 'PehmoGIS-ohjeet' %}

{% trans 'Kyselyyn vastaaminen' %}

{% blocktrans %} Kyselyyn vastaaminen vie aikaa 20 minuutista ylöspäin riippuen vastaustyylistäsi. Toivomme sinun vastaavan kaikkiin kysymyksiin ja karttasivuilla erityisesti tärkeänä pitämiisi asioihin. Kysely etenee vaiheittain, ja antamasi tiedot tallentuvat sitä mukaa, kun etenet kyselyssä. {% endblocktrans %}

{% trans 'Kyselyssä eteneminen' %}

{% blocktrans %} Siirry seuraavaan kysymykseen Jatka-painikkeella. Voit siirtyä takaisinpäin kyselyssä korjaamaan jo antamiasi vastauksia Edellinen-painikkeella tai yläpalkin navigaatiopalkin kautta. {% endblocktrans %}

{% blocktrans %} Kun pääset kyselyn loppuun, etkä enää halua muuttaa vastauksiasi, voit poistua kyselystä sulkemalla selaimen. Jos et ehdi vastata kaikkiin kysymyksiin yhdellä kertaa, voit keskeyttää kyselyyn vastaamisen ja palata myöhemmin täydentämään vastauksiasi (kts. ohje {% endblocktrans %} {% trans 'kyselyn keskeyttämisestä' %}).

{% trans 'Kyselyn keskeyttäminen' %}

{% blocktrans %} Voit jatkaa kyselyä myöhemmin keskeyttämällä vastaamisen Keskeytä-painikkeesta, joka löytyy näkymän oikeasta yläreunasta (kts. kuva oikealla). {% endblocktrans %}

{% blocktrans %} Keskeyttäessäsi kyselyn sinun tulee antaa toimiva sähköpostiosoite. Saat ohjeet ja linkin antamaasi sähköpostiosoitteeseen. Linkki on henkilökohtainen ja sitä kautta voit muuttaa omia vastauksiasi tai jatkaa kyselyn täyttämistä. {% endblocktrans %}

{% trans 'Kartan käyttö' %}

{% trans 'Kartalla liikkuminen' %}

{% blocktrans %} Kartalla liikutaan nuolipainikkeiden (kuva oikealla) avulla. Tee näin: klikkaa haluamaasi suuntaan osoittavaa nuolta hiirellä. Kartta liikkuu yhden pykälän valitsemaasi suuntaan. {% endblocktrans %}

{% blocktrans %} Voit liikkua kartalla myös navigointikartan avulla tai siirtämällä karttaa hiirellä raahaamalla. {% endblocktrans %}

{% blocktrans %} Tee näin raahataksesi karttaa hiirellä: klikkaa hiiren painike alas missä tahansa kohtaa kartan päällä. Pidä painike pohjassa ja siirrä hiirtä haluamaasi suuntaan. Kartta liikkuu hiiren mukana. {% endblocktrans %}

{% trans 'Kartan lähennys ja loitonnus' %}

{% blocktrans %} Kartan lähentämiseen ja loitontamiseen (zoomaamiseen) on kaksi tapaa. Karttaa voi lähentää ja loitontaa zoomauspalkin tai hiiren vierityspainikkeen avulla. {% endblocktrans %}

{% blocktrans %} Zoomauspalkin avulla karttaa voi lähentää klikkaamalla plus-painiketta (+) ja loitontaa klikkaamalla miinus-painiketta. Näkymää voi säätää myös suoraan raahaamalla palkin osoitinta ylöspäin tai alaspäin. {% endblocktrans %}

{% blocktrans %} Jos hiiressäsi on niin sanottu vierityspainike (tai "rulla") voit zoomata myös sen avulla. Kartta zoomaa lähemmäksi, kun vierität rullaa eteenpäin ja taaksepäin, kun vierität rullaa taaksepäin. {% endblocktrans %}

{% trans 'Navigointikartan käyttö' %}

{% blocktrans %} Näkymän oikeasta ylänurkasta löytyy navigointikartta eli pienoiskokoinen kartta, josta näkyy näkymän sijainti suhteessa koko alueeseen. Navigointikartan avulla voi myös liikkua kartalla. {% endblocktrans %}

{% blocktrans %} Näin liikut kartalla navigointikartan avulla: tartu hiirellä kartalla näkyvään punaiseen neliöön ja raahaa sitä hiirellä haluamaasi suuntaan. {% endblocktrans %}

{% trans 'Kartalle merkittyjen kohteiden piilottaminen' %}

{% blocktrans %} Kartalle päällekkäin merkityt kohteet peittävät helposti pohjakartan. Voit halutessasi piilottaa kartalle merkitsemäsi kohteet Piilota omat merkinnät -painikkeen avulla. Painike löytyy toimintopainikkeen yläpuolelta, kuten viereisessä kuvassa on esitetty. {% endblocktrans %}

{% blocktrans %} Kun olet piilottanut omat merkintäsi kartalta, painikkeessa lukee "Näytä omat merkinnät". Saat siis merkintäsi uudelleen näkyviin klikkaamalla painiketta uudemman kerran. {% endblocktrans %}

{% trans 'Kohteiden merkitseminen kartalle' %}

{% blocktrans %} Kyselyssä merkitään kartalle pisteitä, alueita tai viivoja (reittejä). Toimintopainikkeen kuvake kertoo, minkälainen kohde on tarkoitus paikantaa. Esimerkiksi koti merkitään kartalle pisteenä, kulkureitit piirretään viivoina ja naapurusto alueena. {% endblocktrans %}




{% trans 'Kuva yllä: Esimerkit piste-, viiva- ja aluetyökaluista.' %}

{% blocktrans %} Elinympäristön arviointi -osiossa voit merkitä kartalle sekä pisteitä että alueita. Valitse piste- tai aluetyökalu sen mukaan, minkälainen kohteesi on. {% endblocktrans %}


{% trans 'Kuva yllä: Elinympäristön arviointi -osiossa sinun tulee valita joko piste- tai aluetyökalu.' %}

{% trans 'Kodin tai muun pisteen merkitseminen kartalle' %}

{% blocktrans %} Koti merkitään kartalle pisteenä. Muita pistemäisiä kohteita ovat esimerkiksi työ- tai opiskelupaikka, peruspalvelut ja onnen paikka. {% endblocktrans %}

{% trans 'Esimerkki pisteen merkitsemisestä kartalle: Kodin merkitseminen' %}

  1. {% trans 'Etsi aluksi kartalta kotisi sijainti.' %}
  2. {% trans 'Valitse Koti-työkalu' %} {% trans 'klikkaamalla sitä hiirellä.' %}

    {% trans 'Klikatessasi Koti-työkalu aktivoituu ja sen ulkoasu muuttuu:' %}
  3. {% trans 'Klikkaa karttaa kodin kohdalla.' %} {% trans 'Koti sijoitetaan osoittamaasi kohtaan.' %}
  4. {% blocktrans %} Vahvista kodin sijainti avautuvassa ikkunassa klikkaamalla painiketta "Kyllä, tallenna kohde". Mikäli koti meni väärään paikkaan, voit poistaa sen klikkaamalla "Ei, poista kohde". Voit myös siirtää kotia jälkeenpäin, jos huomaat sen olevan väärässä paikassa (kts. {% endblocktrans %} {% trans 'kohteen siirto-ohje' %}).

{% trans 'Reitin piirtäminen kartalle' %}

{% blocktrans %} Reitti (tai viiva) piirretään kartalle hiirellä klikkailemalla. Aloita piirto reitin jommastakummasta päästä. Kun reitti on valmis, lopeta piirto hiiren kaksoisklikkauksella. {% endblocktrans %}

{% trans 'Esimerkki reitin piirtämisestä.' %}

  1. {% trans 'Etsi haluamasi paikka kartalta.' %}
  2. {% trans 'Valitse' %} {% trans 'reitti -työkalu' %}.
  3. {% blocktrans %} Aloita reitin piirto klikkaamalla karttaa reitin alkupään kohdalla. Jatka reitin piirtämistä klikkailemalla karttaa reittiä seuraillen. {% endblocktrans %}
  4. {% trans 'Lopeta piirto hiiren kaksoisklikkauksella.' %}
  5. {% trans 'Kerro reitistä haluamasi asiat ja tallenna reitti.' %}

{% blocktrans %} Huom! Jos reittisi jatkuu näkymän ulkopuolelle, voit siirtää karttaa pitämällä hiiren vasenta nappia pohjassa tai kartan siirtonuolia klikkailemalla. {% endblocktrans %}

{% trans 'Alueen piirtäminen kartalle' %}

{% blocktrans %} Alue merkitään kartalle piirtämällä sen reunaviiva hiirellä klikkailemalla. Aloita piirto klikkaamalla karttaa jossain alueen kulmapisteessä. Jatka piirtämistä klikkailemalla kartalle reunaviiva piste pisteeltä. Kun alue on valmis, lopeta piirto hiiren kaksoisklikkauksella. {% endblocktrans %}

{% trans 'Esimerkki alueen piirtämisestä.' %}

  1. {% trans 'Etsi haluamasi paikka kartalta.' %}
  2. {% trans 'Valitse' %} {% trans 'alue -työkalu' %}.
  3. {% blocktrans %} Aloita alueen piirto klikkaamalla karttaa jossain alueen kulmapisteessä. Jatka alueen piirtämistä klikkailemalla karttaa reunaviivaa seuraillen piste pisteeltä. {% endblocktrans %}
  4. {% trans 'Lopeta piirto hiiren kaksoisklikkauksella.' %}

{% blocktrans %} Huom! Jos reittisi jatkuu näkymän ulkopuolelle, voit siirtää karttaa pitämällä hiiren vasenta nappia pohjassa tai kartan siirtonuolia klikkailemalla. {% endblocktrans %}

{% trans 'Kohteen siirtäminen, muuttaminen tai poistaminen' %}

{% blocktrans %} Voit muuttaa kartalle merkitsemiäsi kohteita valitsemalla kohteen kartalta klikkaamalla. Voit muuttaa sen sisältöä (esimerkiksi kohteesta kirjoittamaasi kuvausta). Voit myös poistaa merkitsemiäsi kohteita kartalta. {% endblocktrans %}

{% trans 'Apua ongelmatilanteisiin' %}

{% trans 'Selaimet ja asetukset' %}

{% blocktrans %} Kysely toimii parhaiten Firefox 3 -selaimella. Muita selaimia, joilla kysely on testattu ja todettu toimivan hyvin ovat IE7, Google Chrome, Opera ja Firefox 2. {% endblocktrans %}

{% trans 'JavaScript ja evästeet on sallittava' %}

{% blocktrans %} PehmoGIS-kysely käyttää JavaScript-kieltä, joten selaimesi tulee sallia sen suorittaminen. Samoin tulee sallia evästeiden käyttö. Evästeet ovat pieniä kiintolevylle tai muistiin tallettuvia tekstitiedostoja, joiden avulla voidaan tallettaa tietoa selaamistasi sivustoista. Evästeiden avulla ei voida käynnistää ohjelmia eikä välittää viruksia koneeseesi. {% endblocktrans %}

{% trans 'Näin muutat selaimen asetukset' %}

{% blocktrans %} Tässä ohjeita yleisimpien www-selainten asetusten muuttamisesta siten, että PehmoGIS-kysely toimii. {% endblocktrans %}

{% trans 'Firefox 2.0 ja 3.0 ja uudemmat:' %}

  1. {% blocktrans %} Valitse valikosta Työkalut > Asetukset. Valitse Asetukset-ikkunassa Sisältö kohta ja valitse kohta "Salli JavaScript" (Firefox 3.0: "Salli JavaScript -komentosarjat"). {% endblocktrans %}
  2. {% blocktrans %} Valitse sitten Asetukset ikkunassa Tietosuoja-sivu. Evästeet-otsikon alta valitse kohta "Sivustot voivat asettaa evästeitä". Hyväksy muutokset painamalla OK. {% endblocktrans %}

{% trans 'Internet Explorer 6.x:' %}

  1. {% trans 'Valitse valikosta Työkalut (Tools) > Internet-asetukset (Internet Options).' %}
  2. {% blocktrans %} Valitse Asetus-ikkunasta Tietosuoja (Security) -välilehti. Aseta tietosuojatasoksi (Security Level) Normaali (Medium tai Medium-low) liukuvalitsinta siirtämällä. {% endblocktrans %}

{% trans 'Internet Explorer 7.0 ja uudemmat:' %}

  1. {% trans 'Valitse valikosta Työkalut (Tools) > Internet-asetukset (Internet Options).' %}
  2. {% blocktrans %} Valitse Internet asetukset -ikkunassa Suojaus (Security) -välilehti. Aseta Vyöhykkeen suojaustaso kohtaan Keskitaso tai Keskivahva. Voit käyttää myös Mukautettua tasoa (Custom Level). Varmista silloin, että ainakin seuraavat kohdat ovat valittuina "Scripting/Active Scripting/Enable". {% endblocktrans %}

{% trans 'Opera:' %}

  1. {% trans 'Valitse valikkoriviltä Tiedosto ja Asetukset...' %}
  2. {% trans 'Valitse uuden ikkunan vasemmasta laidasta kohta Verkko ja sieltä alakohta Yksityisyys.' %}
  3. {% trans 'Valitse Evästeet - kohdasta Ota evästeet käyttöön. Hyväksy asetukset valitsemalla OK.' %}

{% trans 'Tietosuoja' %}

{% blocktrans %} {% endblocktrans %}

{% blocktrans %} Tutkimuksessa ei käsitellä henkilötietolain piiriin kuuluvia tietoja (kuten osoitteita, nimiä tai henkilötunnuksia). Tutkimuksesta on kuitenkin laadittu Tietosuojavaltuutetun ohjeiden mukainen tieteellisen tutkimuksen rekisteriseloste. {% endblocktrans %}

{% blocktrans %} Tutkimusaineiston tietoja käsitellään ryhmittäin, ei yksilötasolla. Tuloksia ryhmitellään esim. sukupuolen, ikäryhmän ja asuinpaikan perustella ja niissä käsitellään ryhmien antamia keskimääräisiä vastauksia, kuten keskimääräistä matkaa tärkeisiin paikkoihin tai kulkutapoja töihin. Julkaistavista tuloksista on mahdotonta jäljittää yksittäisten ihmisten tai yksittäisessä rakennuksessa asuvien ihmisten vastauksia. {% endblocktrans %}

{% blocktrans %} Kysymykset hyvinvoinnista tai sosioekonomisesta asemasta eivät ole tutkimuksen aiheita sinällään, vaan niitä käsitellään ainoastaan kytkettynä tutkimuksen varsinaiseen teemaan, eli elinympäristön kokemuksiin, ja tietoihin yhdyskunnan rakenteellisista piirteistä. {% endblocktrans %}
