{% trans 'Kodin tai muun pisteen merkitseminen kartalle' %}

{% blocktrans %} Koti merkitään kartalle pisteenä. Muita pistemäisiä kohteita ovat esimerkiksi työ- tai opiskelupaikka ja asiointipaikka. {% endblocktrans %}

{% trans 'Esimerkki pisteen merkitsemisestä kartalle: Kodin merkitseminen' %}

  1. {% trans 'Etsi aluksi kartalta kotisi sijainti.' %}
  2. {% trans 'Valitse Koti-työkalu' %} {% trans 'klikkaamalla sitä hiirellä.' %}

    {% trans 'Klikatessasi Koti-työkalu aktivoituu ja sen ulkoasu muuttuu:' %}
  3. {% trans 'Klikkaa karttaa kodin kohdalla.' %} {% trans 'Koti sijoitetaan osoittamaasi kohtaan.' %}
  4. {% blocktrans %} Vahvista kodin sijainti avautuvassa ikkunassa klikkaamalla painiketta "Kyllä, tallenna kohde". Mikäli koti meni väärään paikkaan, voit poistaa sen klikkaamalla "Ei, poista kohde". Voit myös siirtää kotia jälkeenpäin, jos huomaat sen olevan väärässä paikassa (kts. {% endblocktrans %} {% trans 'kohteen siirto-ohje' %}).

{% trans 'Asuinalueiden arviointi' %}

{% blocktrans %} Uudet asuinalueet ja täydennysrakentamiskohteet on merkitty kartalle sinisin monikulmioin ja varustettu talo-merkinnällä. Vasemman reunan "Näytä kaikki"-painikkeesta kartta zoomautuu niin, että kaikki arvioitavat alueet näkyvät näytöllä yhtä aikaa. {% endblocktrans %}
