{% trans 'Kyselyn toteuttaja ja tutkijat' %}

{% blocktrans %} Tutkimus on osa Urbaani Arki -tutkimushanketta. Aalto-yliopiston Yhdyskuntasuunnittelun tutkimus- ja koulutuskeskuksen ja Tampereen teknillisen yliopiston yhteistyökumppanina tutkimuksessa on mm. Tampereen kaupunki. Tutkijoina toimivat FT Marketta Kyttä, FL Kaisa Schmidt-Thomé ja TkL Hanna Kalenoja. {% endblocktrans %}

{% blocktrans %} Urbaani Arki -hankkeessa tutkitaan asukkaiden arkiliikkumista ja arjen palvelujen käyttöä paikkalähtöisesti tunnistaen samalla elinympäristön käytön suhteen erilaisia asukasprofiileja. Arjen sujumisen kannalta on tärkeää, missä asukkaalle keskeiset arkiverkon paikat sijaitsevat ja miten niiden välillä liikutaan. {% endblocktrans %}

{% blocktrans %} Aalto-yliopiston Yhdyskuntasuunnittelun tutkimus- ja koulutuskeskuksen ja Tampereen teknillisen yliopiston tiedonhallinnan ja logistiikan laitoksen yhteishankkeessa tutkitaan eri asukasryhmiä yhteensä kahdeksan kaupungin alueella. Hanketta rahoittavat Tekes ja kaupungit, sekä yrityskumppaneina Ruokakesko, FCG Planeko, Citycon, Fifth Element ja ESRI Finland. Valtionhallintoa hankkeessa edustavat Ympäristöministeriö ja Kaupunkipolitiikan yhteistyöryhmä (Työ- ja elinkeinoministeriö). {% endblocktrans %}

{% trans 'Tietosuoja' %}

{% blocktrans %} Tampereen pehmoGIS-kysely toimii Aalto-Yliopiston suojatulla palvelimella, ja kyselystä koostuvaan tietokantaan on pääsy vain tutkimushankkeessa työskentelevillä henkilöillä. Sovellus ei ole yhteydessä Tampereen kaupungin tietojärjestelmiin. {% endblocktrans %}

{% blocktrans %} Tutkimuksessa ei käsitellä henkilötietolain piiriin kuuluvia tietoja (kuten osoitteita, nimiä tai henkilötunnuksia). Tutkimuksesta on kuitenkin laadittu Tietosuojavaltuutetun ohjeiden mukainen tieteellisen tutkimuksen rekisteriseloste. {% endblocktrans %}

{% blocktrans %} Tutkimusaineiston tietoja käsitellään ryhmittäin, ei yksilötasolla. Tuloksia ryhmitellään esim. sukupuolen, ikäryhmän ja asuinpaikan perustella ja niissä käsitellään ryhmien antamia keskimääräisiä vastauksia, kuten keskimääräistä matkaa tärkeisiin paikkoihin tai kulkutapoja töihin. Julkaistavista tuloksista on mahdotonta jäljittää yksittäisten ihmisten tai yksittäisessä rakennuksessa asuvien ihmisten vastauksia. {% endblocktrans %}
